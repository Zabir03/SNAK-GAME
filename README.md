# 🐍 Snake Game (Web-Based)

A modern and interactive **Snake Game** built using **HTML, CSS, and JavaScript**.  
This project features a responsive UI, smooth gameplay, and dynamic score tracking.

---

## 🚀 Features

- 🎮 Classic Snake Game mechanics  
- 📱 Fully responsive (mobile + desktop support)  
- ⏱ Real-time timer  
- 🏆 High score tracking using Local Storage  
- 🍎 Dynamic food generation  
- 💀 Game over modal screen  
- 🔁 Restart functionality  
- ⚡ Smooth animations and modern UI  

---

## 🛠 Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & responsiveness  
- **JavaScript (Vanilla JS)** – Game logic

## 🎯 How to Play

1. ▶️ Click **Start Game**  
2. ⌨️ Use arrow keys to control the snake:
   - ↑ Up  
   - ↓ Down  
   - ← Left  
   - → Right  
3. 🍎 Eat food to increase score  
4. ❌ Avoid hitting walls  
5. 🔄 Restart after game over
